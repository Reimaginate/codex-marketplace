# Reimaginate Codex Marketplace

Official Codex plugins from Reimaginate.

## Available Plugins

- Design Hub
- Dynamics 365, Dataverse, and Power Pages

## Installation

Install the Design Hub CLI, then run:

```powershell
designhub mcp install codex
```

The installer registers this GitHub-hosted marketplace and installs `design-hub@reimaginate`. Install `dynamics-365-cli@reimaginate` from the same marketplace, then start the local Dynamics 365 MCP server when you need Codex access to Dataverse, Dynamics 365, or Power Pages inventory tools:

```powershell
designhub d365 mcp start
```
